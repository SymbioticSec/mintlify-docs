### Specific tools 

**Guardrails**\
When working on an existing project, the agent first retrieves the repositoryâ€™s security guardrails. These guardrails guide the agent and ensure that any suggested implementation follows company standards and repository best practices. You can define and manage guardrails for a repository in the portal [here](https://app.symbioticsec.ai/guardrails).

**Security Check**\
During feature implementation, the agent may perform a security check by analyzing the code against OWASP security risks. If any risks are identified and not adequately addressed, the agent automatically adjusts the implementation to ensure compliance with security best practices.

**Security Scan**\
Once implementation is complete, the agent runs a security scan using SAST tools to analyze the generated code for potential vulnerabilities. Based on the scan results, the agent iterates on the implementation until no security issues remain.
