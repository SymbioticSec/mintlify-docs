To use Symbiotic Code, please launch the following command in your terminal:

```shellscript
curl ..../install | bash
```

To launch Symbiotic Code, use the command `symbiotic`in your terminal.

## Authenticate 

At the first launch, you will be asked to input your token. Follow steps detailed in the modal to log in.

## Start coding

### Common Commands

`/cd` Use this command to change the directory used by the agent\
`/session` To find your history and switch between sessions
