<Steps>
  <Step title="Downlod the extension">
    You can download the extension at your first login or using this link
  </Step>
  <Step title="Install the extension">
    In the interface, go to `Extensions > … More actions > Install from VSIX…`
    Or you use the command `Extensions: Install from VSIX…`

    Select the file and click on `Install`
  </Step>
  <Step title="Log in">
    Once the installation is finished a notification will ask you to log-in using your Symbiotic Security account.

    <Steps>
      <Step title="Click on Log In" icon="square-arrow-out-up-right">
        ![Extension Log In Notification Pn](/content/devs/images/extension_log_in_notification.png)
      </Step>
      <Step title="Select you token" icon="mouse-pointer-click">
        ![App Select Token Screen Pn](/content/devs/images/app_select_token_screen.png)
      </Step>
      <Step title="Click on Confirm" icon="square-mouse-pointer">
      </Step>
    </Steps>
  </Step>
  <Step title="You're now authenticated" icon="party-popper">
    Your first scan should start in a second or two...
  </Step>
</Steps>
