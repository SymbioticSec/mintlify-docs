---
title: "Share logs"
description: "Share logs with Symbiotic Security"
---

If you’re experiencing some unexpected behavior with our IDE extension, you have the possibility to access logs and share it with us to help us investigate the issue. 

1. In VS Code go to `View > Output`
2. Select `Symbiotic Security`
    
    ![Capture d’écran 2024-10-25 à 15.44.52.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/6215c802-c2a3-46a9-8f21-e2604f4086b4/a9af82d3-f205-4193-ab9b-cf7e00c50f63/Capture_decran_2024-10-25_a_15.44.52.png)
    
3. You have now access to the logs of the extension since you last opened VS Code
    
    ![Capture d’écran 2024-10-25 à 15.45.34.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/6215c802-c2a3-46a9-8f21-e2604f4086b4/ab72f015-e2e7-48fc-ac0f-8ab31ed1d514/Capture_decran_2024-10-25_a_15.45.34.png)
    
4. Click on the three dots and then on `Open in editor` to open the log file
    
    ![Capture d’écran 2024-12-10 à 11.24.22.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/6215c802-c2a3-46a9-8f21-e2604f4086b4/503a0545-4154-413b-b693-4c9a8dd12a5d/Capture_decran_2024-12-10_a_11.24.22.png)
    
5. You can now save this file and share it with us
