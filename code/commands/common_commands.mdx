## Common commands

### `/cd`
Change the working directory for the agent. Start typing the path of the folder and then you can navigate  between suggestions using arrows keys or via your mouse. To select a suggestion, press TAB or click on it. 

### `/session`
View conversation history and switch between sessions. By default, sessions are organized by date then project's folder but you can switch this sort using the TAB key.

### `/new`
Start a new conversation session

### `/models`
Switch between available models.
