---
title: "Common commands"
description: "Common commands for Symbiotic Code"
---

Use `/` or `Ctrl+P` to access the command palette and execute various actions.

- `/init` Initialize the project by creating or updating the agents.md file.
- `/agents` Switch between agent modes (build and plan)
- `/connect` Connect to a provider
- `/exit` Quit the application
- `/help` Show the help dialog
- `/mcp` Select MCPs servers to use
- `/models` Choose wich model you're using
- `/new` Create a new session
- `/review` Launch a subagent to review your code
- `/sessions` Switch between sessions
- `/skills` Load your skills
- `/status` See the status of LSP servers, Formatters and plugins
- `/themes` Switch between themes
